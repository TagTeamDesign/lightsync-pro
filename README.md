== Description ==

**LightSync Pro creates a cloud-to-CMS image pipeline for WordPress and Shopify.**

Connect your creative tools, browse your cloud content inside WordPress, and sync images without downloading, renaming, or re-uploading anything — to WordPress and Shopify simultaneously. When a source image changes, simply re-sync to update it in place — without breaking attachment IDs, layouts, or product images.

Whether you're a photographer publishing from Lightroom, a designer working in Canva or Figma, or a Shopify merchant managing product images — LightSync Pro eliminates the repetitive download-upload cycle with automatic compression and optional AI image generation.

No exports. No duplicate uploads. No broken links.

= How It Works =

1. **Connect** — Authorize Lightroom, Canva, Figma, Dropbox, Shutterstock, or OpenRouter (AI) via secure OAuth
2. **Browse** — View albums, designs, files, folders, licensed images, or generate AI images directly inside WordPress
3. **Choose Destination** — Sync to WordPress, Shopify, or both simultaneously
4. **Sync** — Images are optimized, compressed, and added to your Media Library or Shopify Files

All sources sync to both WordPress and Shopify in the free version — no upgrade required for multi-destination sync.

== Installation ==

1. Upload the `lightsyncpro` folder to `/wp-content/plugins/`  
2. Activate the plugin through the WordPress Plugins menu  
3. Open **LightSync Pro** in the admin sidebar  
4. Connect your first source (Lightroom, Canva, Figma, Dropbox, Shutterstock, or OpenRouter for AI)  
5. Optionally connect Shopify under Sync Destinations  
6. Browse cloud content or generate AI images and click Sync  

= Requirements =

* WordPress 5.8+  
* PHP 7.4+  
* Account with at least one supported platform  
* Shopify store (optional)  

== Frequently Asked Questions ==

= Do I need API keys? =  
No. Secure OAuth authentication is handled through the LightSync broker.

= What happens when I edit an image in Lightroom, Canva, or Figma? =  
Re-sync it. The existing WordPress attachment updates in place without changing the attachment ID or breaking layout references.

= Can I sync to both WordPress and Shopify at the same time? =  
Yes. Choose "Both" as your sync destination.

= Where are my images stored? =  
Images are stored locally in your WordPress Media Library or Shopify Files library. LightSync Pro does not store your images on its servers — the broker only handles authentication and sync coordination.

= Does syncing slow down my site? =  
No. Images are stored locally in your Media Library and optimized with WebP compression.

= How does AI image generation work? =  
Connect OpenRouter, enter a prompt, choose from 400+ AI models, and generate images directly in WordPress. Generated images sync just like cloud sources.

= How does Shutterstock sync work? =  
LightSync Pro syncs images you've already licensed. It does not purchase new licenses.
